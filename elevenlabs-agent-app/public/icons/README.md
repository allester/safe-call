# PWA / visionOS app icons

Add these files for the web app manifest and for building the visionOS app:

- **icon-512.png** — 512×512 px, purpose: any (can have transparency)
- **icon-1024-maskable.png** — 1024×1024 px, purpose: maskable (no transparency; visionOS requires this)

You can use the [WebSpatial sample icons](https://webspatial.dev/assets/files/webspatial-icon-examples-cec65dea794c612f8b689d1d17aac9f3.zip) or generate your own.
